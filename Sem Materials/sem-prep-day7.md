# Types of Plots

## üîπ **Univariate Plots**

‚û°Ô∏è Involve **only one variable** ‚Äî used to study its **distribution, spread, or frequency**.

**Examples:**

| Plot Type       | Description                                                 |
| --------------- | ----------------------------------------------------------- |
| **Histogram**   | Shows distribution of one numeric variable.                 |
| **KDE Plot**    | Smooth version of a histogram (one variable).               |
| **Box Plot**    | Displays spread, median, and outliers for one variable.     |
| **Violin Plot** | Similar to box plot, adds density shape (for one variable). |

üß† _Use when you want to understand how a single variable behaves (e.g., exam scores, ages, prices)._

---

## üîπ **Bivariate Plots**

‚û°Ô∏è Involve **two variables** ‚Äî used to study **relationships or comparisons** between them.

**Examples:**

| Plot Type                                  | Description                                                       |
| ------------------------------------------ | ----------------------------------------------------------------- |
| **Scatter Plot**                           | Shows relationship between two numeric variables.                 |
| **Line Plot**                              | Shows how one variable changes with another (usually time).       |
| **Bar Chart**                              | Compares categorical vs numeric variable.                         |
| **Clustered Bar Chart**                    | Compares two variables side by side for each category.            |
| **Box Plot / Violin Plot (with category)** | When plotted with a category on the x-axis and numeric on y-axis. |

üß† _Use when you want to compare or find correlation (e.g., height vs weight, time vs sales)._

---

## üîπ **Multivariate Plots**

‚û°Ô∏è Involve **three or more variables** ‚Äî used to observe **complex relationships**.

**Examples:**

| Plot Type                      | Description                                                         |
| ------------------------------ | ------------------------------------------------------------------- |
| **Stacked Bar Chart**          | Multiple categories and subcategories (3+ variables).               |
| **Pair Plot**                  | Plots all pairwise relationships among multiple numeric columns.    |
| **Scatter Plot with Hue/Size** | When color or size encodes extra variables (e.g., species, gender). |
| **Violin/Box Plot (with hue)** | Comparing distributions across multiple categories.                 |
| **Subplots**                   | When multiple plots show different variables in one figure.         |

üß† _Use when analyzing multidimensional relationships (e.g., species, sepal length, and petal width in Iris dataset)._
